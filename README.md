LDrawer
=======

Android drawer icon with material design animation

Download
--------

Download [the latest JAR][1] or grab via Maven:
```xml
<dependency>
  <groupId>com.ikimuhendis</groupId>
  <artifactId>ldrawer</artifactId>
  <version>1.0</version>
</dependency>
```
or Gradle:
```groovy
compile 'com.ikimuhendis:ldrawer:1.0'
```
